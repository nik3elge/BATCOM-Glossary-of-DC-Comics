# Глоссарий DC Comics от BATCOM
Глоссарий DC Comics от BATCOM — это комплексный справочник, включающий имена персонажей, их псевдонимы, названия команд, локаций, организаций и других элементов Вселенной DC Comics с переводом с английского на русский. Предназначен для переводчиков, фанатов и исследователей, обеспечивая точное и традиционное соответствие оригинальных названий и удобное использование в контексте комиксов, фильмов и видеоигр.

## Используемые библиотеки

- [List.js](http://listjs.com) — легкая библиотека для сортировки, поиска и фильтрации списков на фронтенде.
