# Глоссарий DC Comics от BATCOM
Комплексный справочник, включающий имена персонажей, их псевдонимы, названия команд, локаций, организаций и других элементов Вселенной DC Comics с переводом с английского на русский. Предназначен для переводчиков, фанатов и исследователей, обеспечивая точное и традиционное соответствие оригинальных названий и удобное использование в контексте комиксов, фильмов и видеоигр.

Этот проект использует List.js (MIT License) — http://listjs.com
